Windows 11 Pro Home Workstation

以最小暴露面提供可審計的遠端存取。

入口由 GitHub Pages 提供，驗證交給 Cloudflare Access，控制平面交給 Cloudflare Worker， 外部路徑只允許 Cloudflare Tunnel。目標主機為 Home Workstation。

Connect Home Workstation Review Security Baseline

Current Mode

Zero-Trust Only

  • No Internet inbound ports
  • Cloudflare Access + MFA required
  • Audit and notifications handled by Worker

若尚未設定 Access 或 Worker API，按鈕會維持占位狀態。

Architecture

固定路徑

GitHub Pages → Cloudflare Access → Worker → Tunnel → RDP Host

Hard Constraints

不可違反

  • 不可直接暴露 RDP 3389
  • 不可使用 port forwarding、UPnP 或 DMZ
  • GitHub Pages 不可保存任何 secret

Primary Path

預設啟動策略

登入成功後，由 Worker 發放短時效 .rdp profile，再交由本機 Microsoft Remote Desktop Client 啟動。
