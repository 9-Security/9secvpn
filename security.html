Minimum Controls

Security Baseline

本頁對齊規格書的最小安全控制與 Windows 11 / 路由器基線，作為 Portal 內的說明頁。

Access

Identity Controls

  • Only approved accounts
  • MFA is mandatory
  • Registered devices preferred
  • Step-up for high-risk sessions

Network

Exposure Rules

  • No Internet inbound allow rules
  • No port forwarding for 3389 or 22
  • No UPnP and no DMZ to workstation
  • Outbound only for Tunnel, DNS and required control traffic

Host

Windows 11 Pro Baseline

  • RDP with NLA enabled
  • Defender + Windows Update enabled
  • PowerShell / process / login audit logging enabled
  • cloudflared runs as Windows Service

Out Of Scope

Do Not Build In This Version
